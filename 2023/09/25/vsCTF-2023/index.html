Web/Optimized Admin Bot

파일 구조는 이러하다.

index.js의 코드는 굉장히 간단한데 /visit 엔드 포인트에서 url을 하나 받아서 그 url의 html소스를 JSDOM으로 연다. /readflag.c가 있는걸로 봐서는 먼가 RCE가 트리거될 것 같이 생겨서 JSDOM에 대해서 간단히 찾아봤더니 runScripts옵션이 dangerously로 설정되어 있으면 javascript 실행을 허용하고 vm을 통해서 컨택스트를 분리한다. 스크립트를 실행할 수 있으니 vm escape를 하면 되는 문제였다.

vm escape는 아래 문서를 참고했다.

https://tipi-hack.github.io/2019/04/14/breizh-jail-calc2.html
